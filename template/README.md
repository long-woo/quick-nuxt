## Build Setup

```bash
# 安装依赖
$ npm install # 或 yarn

# serve with hot reload at localhost:{{serverPort}}
$ npm run dev

# 测试环境，使用 PM2 部署和管理
$ npm run build:test
$ npm run test

# 正式环境，使用 PM2 部署和管理
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

For detailed explanation on how things work, checkout the [Nuxt.js docs](https://github.com/nuxt/nuxt.js).
